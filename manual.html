Manual de Uso
Envio Padrao - Primeira Mensagem (enviar.php)

1. Pre-requisitos

  • PHP 8.0 ou superior instalado
  • Extensao PDO e cURL habilitadas
  • Acesso ao banco de dados (configurado em conexao.php)
  • Acesso a internet (para envio via webhook)

2. Arquivos do projeto

ArquivoDescricao
enviar.phpScript principal - executa todo o fluxo de envio
conexao.phpConfiguracao de conexao com o banco de dados

3. Como executar

Abra o terminal (CMD, PowerShell ou Git Bash) na pasta do projeto e execute:

php enviar.php

O script vai fazer uma serie de perguntas interativas. Veja abaixo cada etapa.

4. Passo a passo das perguntas

5 Total de contatos disponiveis

>> Total de contatos disponiveis: 847 Quantidade para enviar (ou 0 para todos): 200
EntradaO que faz
0Seleciona todos os contatos disponiveis (no exemplo, 847)

O total exibido e a quantidade de contatos unicos por telefone que atendem a todos os filtros selecionados (escola + cursos + aluno + campanha + status nulo).

6 Modo de Execucao

Modo de execucao: [1] ENVIAR (webhook + atualiza status) [2] SIMULAR (apenas gera CSV, nao envia nada) Escolha: 2
OpcaoO que faz
[1] ENVIAREnvia os dados para o webhook (SellFlux) e atualiza o status do contato no banco para selected. Acao irreversivel.
[2] SIMULARApenas consulta o banco e gera o arquivo CSV. Nao envia nada para o webhook e nao altera nenhum dado no banco.
Recomendacao: Sempre execute primeiro no modo SIMULAR para conferir os contatos no CSV antes de fazer o envio real.

7 Salvar CSV (apenas no modo ENVIAR)

Deseja salvar os contatos em CSV? (s/n): s
EntradaO que faz
sGera um arquivo CSV com todos os contatos que serao enviados
nNao gera CSV, vai direto para o envio
No modo SIMULAR, o CSV e gerado automaticamente (esta pergunta nao aparece).

O arquivo CSV e salvo na mesma pasta com o nome:

contatos_Liceu_-_Itaquaquecetuba_2026-02-13_10-30-00.csv

O CSV usa separador ponto e virgula (;) e abre corretamente no Excel com acentos.

5. Durante a execucao

Modo ENVIAR

Para cada contato enviado com sucesso, o script:

  1. Envia os dados para o webhook do SellFlux
  2. Atualiza o status do contato no banco para selected
  3. Aguarda 1 segundo antes do proximo envio

Modo SIMULAR

8. Dicas importantes

Sempre simule primeiro: Use o modo SIMULAR para gerar o CSV e conferir os contatos antes de fazer o envio real.
Contatos ja enviados nao aparecem de novo: Apos o envio, o status do contato muda para selected. Na proxima execucao, ele nao sera mais listado (o script filtra apenas status IS NULL).
Intervalo entre envios: O script aguarda 1 segundo entre cada envio para nao sobrecarregar o webhook.
Cancelar durante o envio: Pressione Ctrl+C no terminal para interromper. Os contatos ja enviados terao seu status atualizado no banco.
